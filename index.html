Frequently Asked Questions

Does this connect to my Stripe account?

No. You export a CSV file from your Stripe dashboard and upload it here. No OAuth, no permissions, no security risk. Your data never touches a database — files are processed in memory and deleted immediately.

Does it work with QuickBooks Desktop or only Online?

Currently optimized for QuickBooks Online CSV import. QuickBooks Desktop users can import the same file via the IIF/CSV import feature — contact support if you need a custom format.

What about refunds and disputes?

Refunds in your Stripe export appear as negative amounts. We include them in the conversion as negative revenue entries. Disputes and chargebacks are also handled — they show up as separate fee line items.

Is my data safe?

Files are processed entirely in memory and never stored on disk or in a database. After conversion, the file is discarded. We don't keep copies, backups, or logs of your financial data.
