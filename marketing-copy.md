## PRZYKŁADOWE FAQ (na stronę)

**Q: Does this connect to my Stripe account?**
A: No. You export a CSV file from your Stripe dashboard and upload it. No OAuth, no permissions needed.

**Q: Does it work with QuickBooks Desktop?**
A: Currently optimized for QuickBooks Online. Desktop users can import the file via the IIF/CSV import — contact us if you need a custom format.

**Q: What about multi-currency?**
A: The tool preserves currency information from your Stripe export. You may need to set up multi-currency in QuickBooks separately.

**Q: Is my data safe?**
A: Files are processed in memory and never stored on disk. After conversion, the file is discarded immediately.
